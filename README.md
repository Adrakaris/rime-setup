# 谷歌五笔增加

以原本RIME五笔86版为基础

修改更顺18030版，主要改变如谷歌五笔一样：
* 母亦有「XYYI」
* 象中从「倒日J」变成「口K」同免等字
* 兆亦有「QI」先取儿部

## 特殊符号：

> 为了写特殊符号该五笔允许五字码（所以变成六笔），五字码惟能输入特殊符号，无汉字用五安码

东方符号：
* 注意：zzzz + 相等拼音第一字母

欧洲符号：
* 希腊语：zzxl + 相等拉丁第一字母〔xl=希腊〕
* 俄语：zzcy + 相等拉丁第一字母〔cy=cyrillic〕

数字等无类符号：
* 圈数 ① ：zzqs + a-j 〔qs=圈数〕
* 括号数 ㈠ ：zzks + a-j〔ks=括(号)数〕
* 罗马数 ⅰ ：zzlm + a-j〔lm=罗马〕
